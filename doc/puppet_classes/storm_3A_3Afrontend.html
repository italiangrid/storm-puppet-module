  • db_host (String) (defaults to: $storm::frontend::params::db_host) —

    Host for database connection. Default is localhost.

  • db_user (String) (defaults to: $storm::frontend::params::db_user) —

  • db_passwd (String) (defaults to: $storm::frontend::params::db_passwd) —

  • port (Integer) (defaults to: $storm::frontend::params::port) —

    Frontend service port. Default is 8444.

  • threadpool_maxpending (Integer) (defaults to: $storm::frontend::params::threadpool_maxpending) —

    Size of the internal queue used to maintain SRM tasks in case there are no free worker threads. Default is 200

  • threadpool_threads_number (Integer) (defaults to: $storm::frontend::params::threadpool_threads_number) —

    Size of the worker thread pool. Default is 50.

  • gsoap_maxpending (Integer) (defaults to: $storm::frontend::params::gsoap_maxpending) —

    Size of the GSOAP queue used to maintain pending SRM requests. Default is 1000.

  • be_xmlrpc_host (String) (defaults to: $storm::frontend::params::be_xmlrpc_host) —

    Backend hostname. Default is localhost.

  • be_xmlrpc_port (Integer) (defaults to: $storm::frontend::params::be_xmlrpc_port) —

    Backend XML-RPC server port. Default is 8080.

  • be_xmlrpc_token (String) (defaults to: $storm::frontend::params::be_xmlrpc_token) —

    Token used for communicating with Backend service. Mandatory, has no default.

  • be_xmlrpc_path (String) (defaults to: $storm::frontend::params::be_xmlrpc_path) —

    XML-RPC server path. Default is /RPC2.

  • be_recalltable_port (Integer) (defaults to: $storm::frontend::params::be_recalltable_port) —

    REST server port running on the Backend machine. Default is 9998.

  • check_user_blacklisting (Boolean) (defaults to: $storm::frontend::params::check_user_blacklisting) —

    Enable/disable user blacklisting. Default is false.

  • argus_pepd_endpoint (String) (defaults to: $storm::frontend::params::argus_pepd_endpoint) —

    The complete service endpoint of Argus PEP server. Mandatory if check_user_blacklisting is true.

  • monitoring_enabled (Boolean) (defaults to: $storm::frontend::params::monitoring_enabled) —

    Enable/disable monitoring. Default is true.

  • monitoring_time_interval (Integer) (defaults to: $storm::frontend::params::monitoring_time_interval) —

    Time interval in seconds between each monitoring round. Default is 60.

  • monitoring_detailed (Boolean) (defaults to: $storm::frontend::params::monitoring_detailed) —

    Enable/disable detailed monitoring. Default is false.

  • security_enable_mapping (Boolean) (defaults to: $storm::frontend::params::security_enable_mapping) —

    Flag to enable/disable DN-to-userid mapping via gridmap-file. Default is false.

  • security_enable_vomscheck (Boolean) (defaults to: $storm::frontend::params::security_enable_vomscheck) —

    Flag to enable/disable checking proxy VOMS credentials. Default is true.

  • log_debuglevel (String) (defaults to: $storm::frontend::params::log_debuglevel) —

    Logging level. Possible values are: ERROR, WARN, INFO, DEBUG, DEBUG2. Default is INFO

  • gridmap_dir (String) (defaults to: $storm::frontend::params::gridmap_dir) —

    Gridmap directory path. Defailt value is: /etc/grid-security/gridmapdir

  • gridmap_file (String) (defaults to: $storm::frontend::params::gridmap_file) —

    Gridmap file path. Defailt value is: /etc/grid-security/grid-mapfile
